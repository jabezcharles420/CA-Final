# CA Final — Quick Reference Guides

Web-verified quick-reference revision guides for five CA Final subjects, built from OCR'd
concept notes and cross-checked against ICAI / statutory sources (Ind AS, CGST/Customs, the
Income-tax Act, Standards on Auditing, SEBI/RBI).

- **Live site:** https://jabezcharles420.github.io/CA-Final/
- Search subjects/topics from the hub, or open a guide directly and search within it.
- Light/dark theme toggle, persisted per browser.
- Each guide is a single self-contained HTML file (works offline, phone + laptop).
